# 综述
XTab

> 基于XScroll的移动端tab组件

## Version

2.0.0

## Demo

[simple](../demo/index.html)

[effect](../demo/effect.html)

## Doc

* `config`：
    * `renderTo` 渲染节点
    * `scrollDelay` 延迟触发滚动，默认为0
    * `onShow` 自定义弹层弹出
    * `onHide` 自定义弹层隐藏
* `render()` 渲染
* `switchTo(index)` 选中第index个tab
* `slideDown()` 弹层弹出
* `slideUp()` 弹层隐藏
* `toggleSide()` 自动切换弹层弹出隐藏
